# Proxizen

Une application web permettant d'accompagner les aidans familiaux.

http://152.77.90.121/

## Technologies utilisés

- HTML+CSS
- PHP
- JavaScript
- TypeScript
- React
- Vite
- PHPUnit
- Laravel

# Installation

1. Clonez le dépôt:
```shell
git clone https://gricad-gitlab.univ-grenoble-alpes.fr/iut2-info-stud/2025-s3/s3.01/team-15/proxizen.git
```
2. Installez les paquets npm
```
npm install
```
3. Compilez le site
```
npm run build
```
4. Lancez le site avec composer
```
composer run dev
```
